# KUSANAGI Runs on Docker
KUSANAGI Runs on Docker (hereinafter RoD) provides the functions of KUSANAGI using Docker compose.

## Requires
### Require OS
- CentOS7
- Ubuntu18.04
- Windows10(WSL and docker-machine(VirtualBox or Other OS))
- Mac(with Docker for mac)

### Requre packages.
- bash
- git
- sed
- awk
- grep
- gettext
- envsubst
- curl
- docker(over 18.09)
- docker-compose
- docker-machine
